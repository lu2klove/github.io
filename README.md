# 🎰 Lotto Winning Simulator

Python을 활용한 로또 당첨 확률 시뮬레이터입니다. 단순 번호 생성을 넘어 수만 번의 시뮬레이션을 통해 로또의 확률적 기댓값을 시각적으로 확인할 수 있습니다.

## ✨ Key Features
- **Random Generation**: `random.sample`을 활용한 비복원 추출 방식 구현.
- **Rank Logic**: 1등부터 5등까지의 당첨 조건 정밀 구현 (보너스 번호 포함).
- **Batch Simulation**: 대량의 자동 구매 시뮬레이션 및 통계 산출.

python main.py
